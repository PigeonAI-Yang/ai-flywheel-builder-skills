# Friction Capture

## Role

Find raw material in real practice.

The user often thinks "this is just messy work". Treat messy repeated work as potential artifact material.

## Extract

For each friction, identify:

- concrete situation
- repeated pain
- AI failure mode
- manual workaround
- what can be public
- possible artifact
- possible content angle

## Output

Match the user's language in output. For Chinese users, use:

```text
摩擦：
重复模式：
Artifact 候选：
公开安全角度：
下一步最小动作：
```

For English users, use:

```text
Friction:
Pattern:
Artifact candidate:
Public-safe angle:
Next smallest action:
```

## Reject

Do not keep:

- pure emotion with no event
- one-off annoyance
- sensitive details that cannot be abstracted
- ideas not grounded in practice
